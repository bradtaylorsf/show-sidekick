# Script Director - Character Animation Pipeline

## When To Use

Use this stage after proposal approval. Write only action beats, dialogue, captions, and scene sections that the planned rig can plausibly perform.

## Action-Led Writing

Every section should include the character's visible job: action, reaction, expression, prop interaction, or pose hold. Avoid writing emotional states that have no visual acting equivalent.

## Process

1. Split the story into timed sections with start and end estimates.
2. For each section, name the character, action, emotion, and any prop.
3. Keep dialogue short enough for the acting to read.
4. Mark actions that require special rig parts, pivots, mouth shapes, or attachment points.
5. Preserve the approved runtime and delivery promise in notes.

## Quality Gate

- sections are timed,
- required actions and emotions are visible,
- action beats can become action_timeline entries,
- dialogue does not crowd the acting.
