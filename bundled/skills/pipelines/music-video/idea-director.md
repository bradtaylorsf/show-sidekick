# Idea Director - Music Video Pipeline

## When To Use

Use this stage after source review confirms the track and lyric timing. The idea stage locks the visual benchmark and production constraints before script and scene planning.

## Locked Defaults

- Canvas: `1080×1920 vertical (9:16)`.
- Maximum scene duration: `5.0 seconds`.
- Sample-first threshold: `$0.50` or `15 min`.
- Beat-drop hype tag placement: `1.5-2 sec` before first vocal.

Sample-first is not optional for any production estimated > $0.50 or > 15 min.

## Process

1. Write a brief with title, audience, platform, tone, duration, hook, and key points.
2. Identify one accent color per major character or concept.
3. Name concept sections with actual terms, not generic verse labels.
4. State how Brad's reference music-video acts as the visual benchmark.
5. Record when the sample-first gate must happen.

## Quality Gate

- brief validates,
- canvas is 1080×1920 vertical (9:16),
- scene duration cap is 5.0 seconds,
- visual benchmark and sample-first trigger are explicit.
