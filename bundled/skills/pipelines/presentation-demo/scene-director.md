# Presentation Demo Scene Director

## Slide Mapping

Every narrated script section must map to one or more `deck_manifest.slides[].id` values or a declared support visual. Keep scene order aligned to narration timing unless the approved proposal says to reorder slides.

## Scene Treatment Catalog

Use `slide_image` scenes for source slide screenshots. Express zoom/pan, highlight rectangles, callouts, captions, and support visuals as scene requirements so edit decisions can render them deterministically.

## Static Slideshow Risk

Flag any run where most scenes are unchanged slide screenshots. A valid presentation-demo needs visible motion or overlays on slide scenes, plus readable text at the delivery size.
