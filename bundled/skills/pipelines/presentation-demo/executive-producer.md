# Presentation Demo Executive Producer

## Pipeline State Machine

Run stages in manifest order: idea, capture, proposal, script, cuesheet, scene_plan, assets, edit, compose, publish. The deck is the source visual artifact, voiceover is the master clock, and the final delivery promise is an animated explainer/demo rather than a static slide playback.

## Mandatory Locked Decisions

Lock the deck source, aspect ratio, output target, renderer family, and render runtime before compose. Remotion is the deterministic default. HyperFrames is allowed only when the approved proposal or edit decisions explicitly require it for a specific treatment. A silent runtime swap is a CRITICAL governance violation.

## When To Stop And Check With The Human

Stop for human approval before paid TTS, generated support visuals, source-deck transformations that change meaning, or any downgrade from animated demo to static slideshow. Surface authenticated online deck links as blockers unless the operator provides a downloadable source.
