# Show Roadmap

Use this as a lightweight planning board for the shows in this project.

## Three Layers

| Layer | Owns | Plan here |
|---|---|---|
| 1. Brand / show | Durable identity, audience, recurring cast, publishing lane, defaults | What show are we building and why? |
| 2. Pipeline manifests | Workflow shape, stages, tool capabilities, approval gates, output format | Which production recipe does each show type use? |
| 3. Stage-director skills | How each stage should think, inspect inputs, call tools, review, and revise | What instructions make the pipeline feel specific to this show? |
