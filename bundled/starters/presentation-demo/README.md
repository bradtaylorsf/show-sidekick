# Presentation Demo Starter

General deck-to-demo starter for validating the `presentation-demo` pipeline with a safe local fixture deck.

- Starter: `presentation-demo`
- Pipeline slug: `presentation-demo`
- Playbook: `flat-motion-graphics`
- Fixture size: 1231 bytes
- Expected sample duration: 18 seconds
- Expected aspect: 16:9
- Sample support: zero-key
- Sample inputs: `deck.pdf`, `script.txt`, and `notes.txt`

The sample episode is pre-filled for `showkick init --starter presentation-demo` and uses project-relative input paths under `shows/presentation-demo/inputs/sample-episode/`. Replace `deck.pdf` with a PDF, PPT, PPTX, or direct downloadable deck source in your own episode file.
