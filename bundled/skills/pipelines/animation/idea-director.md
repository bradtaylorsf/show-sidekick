# Idea Director - Animation Pipeline

## When To Use

Use this stage to define what the animated piece should accomplish and how complex it needs to be.

## Shared Visual Contract

Use `bundled/skills/_shared/video-prompting.md` when describing generated shots or reference-inspired visual intent.

## Process

1. Identify the audience, platform, duration, tone, and animation promise.
2. Choose a style lane: kinetic type, abstract motion, diagram animation, Lottie/vector, character-free narrative, or hybrid motion graphics.
3. Name the simplest credible runtime direction, without locking it yet.
4. Capture constraints: source assets, brand rules, captions, VO, music, and export needs.

## Output Contract

Produce a schema-valid `brief` and decision-log entry describing the selected animation promise.
