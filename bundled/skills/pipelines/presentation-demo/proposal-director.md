# Presentation Demo Proposal Director

## Treatment Options

Offer deck-aware treatments that combine slide screenshots with zoom/pan, highlighted regions, callouts, captions, and optional support visuals. Reject plain static slideshow playback as a delivery-promise downgrade.

## Runtime Lock

Set `production_plan.render_runtime` and `edit_decisions.render_runtime` expectations. Remotion is the deterministic default. HyperFrames must be explicitly justified and approved in the proposal before compose.

## Approval Gate

The human must approve treatment, script direction, narration provider, and any paid generation before cuesheet/TTS work begins. Record provider and cost expectations in the decision log.
