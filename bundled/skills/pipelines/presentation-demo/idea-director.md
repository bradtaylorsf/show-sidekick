# Presentation Demo Idea Director

## Deck Demo Promise

Classify the deck source as local PDF, PPT, PPTX, or direct downloadable URL. Record audience, use case, desired duration, aspect ratio, and export target. State clearly that the output is an animated explainer/demo with slide motion, callouts, highlights, captions, and narration timing.

## Input Rules

Prefer local files. For online slide links, continue only when the URL is directly downloadable without browser auth. Preserve source provenance and identify whether speaker notes, slide text, or operator notes should drive the script.

## Runtime Preference

Use Remotion by default. Record HyperFrames only when the operator approves a specific HTML/GSAP treatment that cannot be represented cleanly in Remotion.
