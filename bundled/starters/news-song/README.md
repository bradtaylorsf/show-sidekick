# News Song Starter

Audio-led news-song starter for short sourced or source-ready musical explainers.

- Starter: `news-song`
- Pipeline slug: `news-song`
- Playbook: `news-song`
- Fixture size: 120224 bytes
- Expected sample duration: 15 seconds
- Sample inputs: synthesized `track.wav` and `lyrics.txt`

The sample episode is pre-filled for `showkick init --starter news-song` and uses project-relative input paths under `shows/news-song/inputs/sample-episode/`.
