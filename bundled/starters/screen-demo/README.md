# Screen Demo Starter

General screen-demo starter for short synthetic terminal walkthroughs.

- Starter: `screen-demo`
- Pipeline slug: `screen-demo`
- Playbook: `clean-professional`
- Fixture size: 847 bytes
- Expected sample duration: 15 seconds
- Sample inputs: `terminal-steps.yaml`, `narration.txt`, and `notes.txt`

The sample episode is pre-filled for `showkick init --starter screen-demo` and uses project-relative input paths under `shows/screen-demo/inputs/sample-episode/`.
