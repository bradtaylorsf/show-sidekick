# Presentation Demo Starter

General deck-to-demo starter for short animated explainer rough cuts from safe deck fixtures.

- Starter: `presentation-demo`
- Pipeline slug: `presentation-demo`
- Playbook: `clean-professional`
- Fixture size: 1360 bytes
- Expected sample duration: 20 seconds
- Expected aspect: `16:9`
- Sample support: zero-key `false`, paid-demo `true`
- Sample inputs: `deck.pdf`, `operator-notes.txt`, voice preference, duration, and aspect

This starter provides fixture-backed setup and a paid-demo sample path. It requires the paid-demo provider profile for narration and render validation.
