# Idea Director - Documentary Montage Pipeline

## When To Use

Use this stage to turn the user's topic into a retrieval-first montage brief. There is no proposal or script stage in this pipeline, so the brief must carry the creative promise, source strategy, and no-narration decision clearly.

## Briefing Rules

Capture the archive theme, era, places, people or institutions, source families, length target, aspect ratio, mood, and any source restrictions. Name at least three search directions that can grow the clip corpus.

No narration unless the user explicitly asks. Adding voice is a MAJOR change and requires user approval per the Decision Communication Contract.

If the user explicitly asks for voice, stop and apply the Decision Communication Contract before writing a narration-led brief. Do not silently turn the pipeline into a voiceover documentary.

## Process

1. Write a concise `brief` with title, audience, platform, tone, duration, hook, and key points.
2. Record the montage's source families: public domain archive, newsreel, institutional footage, stock video, still-image pan/scan, maps, or user-supplied media.
3. Define what retrieval must prove before the scene plan can proceed.
4. Keep the creative promise visual-led: sequence, contrast, captions, source labels, music bed, and end tag.
5. Start the decision log with no-narration state, rights posture, and retrieval strategy.

## Quality Gate

- the brief is visual-led and retrieval-first,
- source families are searchable,
- no narration is added without explicit approval,
- the target length and platform are explicit,
- the scene planner has enough seed queries to build a corpus.
