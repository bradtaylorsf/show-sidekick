# Show Types

This catalog is the public index of show types Show Sidekick can create. SR-7 owns the full validation matrix; this stub keeps the public README link live while the catalog is expanded.

## Current Families

| Show type | Good for | Starter or pipeline |
|---|---|---|
| Animated explainer | First no-key video, product concepts, educational shorts | `animated-explainer` |
| Music video | Audio-led videos that cut to beats, sections, and lyrics | `music-video` |
| News song | Source-backed musical commentary and fast visual explainers | `news-song` |
| Product or workflow demo | Screen-led walkthroughs and launch demos | `screen-demo`, `product-demo` |
| Cinematic trailer | Motion-led image-to-video promos and story trailers | `cinematic` |
| Talking head or avatar | Narration-led presenter videos | `talking-head`, `avatar` |
| Hybrid source/generated video | Source clips with generated support scenes | `hybrid` |
| Localization dub | Translated or dubbed variants of existing episodes | `localization-dub` |
| Clip factory | Short clips from longer source material | `clip-factory` |

Agents should run `showkick ls starters` and `showkick ls pipelines` in the project before promising a specific path, because local projects can override or add show types.
