# Rave Queen Starter

Show starter for a Rave Queen cinematic demo brief. Rave Queen is not a bundled pipeline type; it is a show starter layered on a real pipeline.

- Starter: `rave-queen`
- Pipeline slug: `cinematic`
- Playbook: `anime-ghibli`
- Fixture size: generated from `inputs/sample-episode/`
- Expected sample duration: 15 seconds
- Sample inputs: `brief.txt`

## Pipeline Decision

Picked `cinematic` because the starter is a mood-forward show demo with trailer pacing, title-card beats, and reference-led visuals. Rejected `animation` for the default binding because it is better for narrated motion systems and stylized explainers; Rave Queen can still use that pipeline later as a user-owned show override if the brief becomes a pure animated sequence.
