# Music Video Starter

Audio-led vertical music video starter for short beat-synced rough cuts.

- Starter: `music-video`
- Pipeline slug: `music-video`
- Playbook: `beat-synced-lyric-video`
- Fixture size: 120212 bytes
- Expected sample duration: 15 seconds
- Sample inputs: synthesized `track.wav` and `lyrics.txt`

The sample episode is pre-filled for `showkick init --starter music-video` and uses project-relative input paths under `shows/music-video/inputs/sample-episode/`.
